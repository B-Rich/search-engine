## Introduction

The Jieba Chinese Word Segmentation Implemented By C++ .

## Usage 

### Dependencies

+ `g++ (version >= 4.1 is recommended) or clang++`;
+ `cmake (version >= 2.6 is recommended)`;

### Download & Compile

```sh
git clone --depth=10 --branch=master git://github.com/yanyiwu/cppjieba.git
cd cppjieba
mkdir build
cd build
cmake ..
make
```

### Unit Testing

```
make test
```
